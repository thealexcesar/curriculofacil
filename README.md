# Currículo Fácil

A free, accessible resume builder that runs entirely in the browser.
No login. No data collection. No server.

## Features
- Step-by-step guided form
- Real-time PDF-faithful preview
- Auto-save to localStorage
- Mobile-first
- Print / Save as PDF

## Tech Stack
HTML · CSS (custom properties) · Vanilla JS (ES modules)
