# Currículo Fácil — Easy Resume

A free, accessible resume builder that runs entirely in the browser.
No login. No server. No data collection.

## Features
- Step-by-step guided form (5 steps)
- Real-time CV preview
- i18n support (pt-BR, en, de)
- Auto-save to localStorage
- Print / Save as PDF
- Mobile-first

## Tech Stack
HTML · CSS (custom properties) · Vanilla JS (ES Modules)

No build tools. No frameworks. Works directly on GitHub Pages.
